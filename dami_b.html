Project Description:

Divine Beauty & Aesthetics is an interactive website that explores the history and cultural significance of Black hairstyles. Built with HTML, CSS, and JavaScript, it features a dynamic timeline and a searchable gallery using original JSON data. Designed with a luxury aesthetic, the site is fully responsive and offers an engaging way to discover Black hair heritage across different eras.
